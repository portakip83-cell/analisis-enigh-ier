# Dashboard de Análisis ENIGH - IER UNAM

Este repositorio contiene una aplicación interactiva desarrollada en **Streamlit** para el análisis multivariado del consumo energético en hogares mexicanos, basada en los microdatos de la ENIGH.

## Características
- **Exploración Dinámica:** Filtros por año y selección de variables de la ENIGH.
- **PCA:** Análisis de componentes principales con perspectiva de género.
- **Análisis de Redes:** Visualización de la interconexión entre variables socioeconómicas y energéticas.

## Requisitos
El sistema requiere Python 3.9+ y las librerías listadas en `requirements.txt`.
